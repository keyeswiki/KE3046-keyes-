# 常见故障排除

## 1 上传代码后小车轮子疯狂转动

答：1.请检查左右的电机是否接错了接口。

## 2 电脑识别不了USB端口

答：1.请确保已参考‘开发环境配置’中驱动安装成功。

2.  请检查USB线是否良好。

3.  更换电脑USB接口

## 3 上传代码不成功

答：1.请检查Arduino IDE
‘工具栏’下开发板和端口是否配置正确（端口可以从设备管理器中查看到）

2.检查扩展板上的蓝牙开关是否是on挡位，上传代码的时候开关必须是off档，否则上传不成功。

## 4 超声波无法测距

答：禁止将超超声波的vcc和gnd接反，否则可能会烧坏超声波传感器，，超声波无法测距请仔细检查接线是否对应</span>
